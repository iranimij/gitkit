This plugin helps you to upload your plugins by git repository links.

== Description ==

This plugin is kind of Git updater, It means that you can update your site with new plugins just by their repository links. you just need to go to the 'Add Plugins' page
and then put the git repository link, After that you should define which branch do you want to clone.

== Who is it for? ==
Anyone who need to upload their plugins or themes with github repository.

== In the future, we will add some useful features like ==

* You will be able to change the branch.
* Theme uploading feature.
* Updating plugin or theme feature.

== Installation ==

Upload the "GitKit" to plugin directory and Activate it.
Before starting please check the settings page, "Settings > GitKit" and save your Github access token.
